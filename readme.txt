Welcome to Fletchers Calculator

This is a simple To-Do list app that keeps track of your current tasks! Simply add or remove tasks from your
daily To-Do list by following the prompts when given! From the main menu you can add, remove, and view your 
tasks. 



Navigation:
1. Add a task     -->  Choose to ADD a task
2. Delete a task  -->  Choose to DELETE a task
3. View tasks     -->  Choose to VIEW your tasks
4. Quit           -->  Choose to exit the application

When addind and deleting tasks you will asked if you would like to add or delete more. 
When prompted please use "Y" to keep adding or deleting and "N" if you are finished.
